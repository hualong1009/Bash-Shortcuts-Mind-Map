# 简介

   我创建了一个仓储，主要是通过思维导图的方式介绍一些Bash终端的快捷键，仓储总包含思维导图的源文件，以及导出的一张PNG图片，用法均是在linux下的`bash` 4.2.46做了验证，可以保证文中的快捷键真实有效。基于仓储中的用法，可以更加高效快捷地使用`bash`工具。

   仓储中包含一个思维导图源文件，思维导图导出的PNG图片，以及README.md和LISENCE.md
